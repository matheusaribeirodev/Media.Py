# Media.Py
📊 Projeto Calculadora de Média

Uma aplicação simples desenvolvida para calcular a média das notas de um aluno de forma rápida e prática. O projeto tem como objetivo praticar conceitos básicos de programação, como variáveis, operadores matemáticos e estruturas condicionais.

🚀 Funcionalidades
Entrada de múltiplas notas
Cálculo automático da média
Exibição do resultado final
Verificação de aprovação ou reprovação
Interface simples e intuitiva
